From Protocol to x402 Revenue

Phase 1 — Week 1-2
MLP-CORE Implementation
Envelope + Blob + User signature. Basic storage on IPFS via Pinata. Simple retrieval API. No fancy features—just the minimal viable protocol.
Phase 2 — Week 3-4
x402 Endpoint
Wrap the retrieval API with x402 payment headers. Price per read: $0.001. Register with Coinbase Bazaar for discovery. First revenue.
Phase 3 — Month 2
MCP Integration
Publish as MCP server. Any Claude/GPT agent can now "remember" across sessions by paying for memory reads. Protocol adoption begins.
Phase 4 — Month 3+
Polyphonic Integration
Full implementation in Polyphonic. Multi-model shared memory. The showcase for what MLP enables. The wedge becomes the platform.
