Design System v1.0

Polyphonic Grayscale Spectrum

A comprehensive 24-shade grayscale system engineered for premium dark interface design. Every value is intentional. Every relationship creates depth.

01

Canvas Depths

The foundational blacks. These form the deepest layer of your interface — the void from which everything else emerges.

#0a0a0a
4% lift. Sidebar and panel backgrounds.

Text Hierarchy

Eight levels of text prominence. Combined with font-weight, this creates nuanced information architecture.

Primary — Headlines, key values, focus points
#ffffff · text-primary
Secondary — Subheadlines, emphasis text
#e8e8e8 · text-secondary
Body — Standard body copy, descriptions
#c0c0c0 · text-body
Tertiary — Supporting text, labels
#909090 · text-tertiary
Quaternary — Meta info, timestamps
#686868 · text-quaternary
Dim — Placeholders, hints
#484848 · text-dim
Muted — Disabled states, decorative
#343434 · text-muted
Ghost — Watermarks, barely visible
#242424 · text-ghost
